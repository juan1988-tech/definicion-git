¿Que es GIT?

Git es un sistema de control de versiones de proyectos tecnologicos en entornos colaboratvos, funciona a través de la linea de comandos, con los cuales puedes crear repositorios, que son archivos de trabajo los cuales puedes modificar, crear, editar, tanto en tu propio espacio de trabajo como en el espacio de trabajo de tu equipo. Practica cualesquier proyecto tecnologico; independientemente que sea de pogramación, diseño, diseño web, etc. Usa git hub para el control de tus versiones

¿Que es GIT HUB?

GIt Hub es la pagina oficial de Git, es un sitio web en el cual puedes subir tus repositorios para ser modificados de forma remota por el equipo de trabajo. Git hub se conoce en el mundo como la red social de los programadores tanto programadores como diseñadores, o cualquier profesional de la tecnología puede subir el código del proyecto. Para así mostrar tu portafolio, y fortalecer tu marca personal

¿Por que deberías usar git y git hub

Debes aprender a usar Git y Git Hub para aprender a trabajar en entornos colaborativos, todo programador, desarrollador web, creativo, estudiante, product manager trabaja constantemente en ambientes colaborativos que requieren de control de versiones. En tiempos de Coronavirus, el trabajo remoto se convierte en un imperativo para la supervivencia de las empresas. Aprende a crear, modificar, destruir, y administrar repositorios. Esto será una competencia clave en tu desarrollo profesional
